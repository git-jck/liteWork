NOTE 1: width auto and margin auto - margin auto or ml or mr only works if there is a specific with in px or in %. But in flex margin auto is also work if there is no specific width or not mention width at all.

NOTE 2: display: inline to ul - you dont have to apply list-style: none, its automaticallly remove the list bullets, and also remove the space accquire by bullets.

NOTE 3: flex effect - by applying flex, child element behave like a block element, even if its inline or given display: inline. means it takes width and height.

NOTE 4: flex width : auto - in flex width auto only take the area required by the content, or you give the width, without flex it takes the whole area. so avoid using flex inside flex.
